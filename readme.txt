== Description ==

A widget that parse a World Of Warcraft character's feed and link achievements and items to wowhead.com


== Installation ==

1. 	Unzip the archive to a directory of your choice
2. 	Upload the contents of the directory to wp-content/plugins/
3. 	Activate the plugin
4. 	Add the widget to you sidebar in Appearance -> Widgets (your theme need to be widgetized http://automattic.com/code/widgets/themes/)
5. 	Configure your widget:
Title: The title you want for your widget.
Realm: The server you're playing on.
Toon's name: The name of your character.
Limite:	How many activity you want to show.
Us or eu: The continent you're playing on (related to the armory: eu.wowarmory.com or us.wowarmory.com other should work too)
6. 	Enjoy !
